Description in Wikipedia

Phone Number conversion

This should become clickable phone number links:
  • Tel: +49-1234-5678-9012
  • Fax: (089) 1234 5678-90
  • Mobile: 001 414 1234 4321
  • French: 23.45.67.89
This should NOT be phone numbers:
  • 2012-09-03 12:00:00
  • 123456789
